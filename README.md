### ✨ What it does

Sidekick is the operating system for future founders.

- **Write** your plan across 11 focused sections
- **See** your plan as a visual canvas
- **Ship** your plan as a beautiful PDF document
- **Ask** section-specific AIs for help at every step

Powered by Google Gemini. Designed like Vogue meets Linear. Built with soul.

---

### 🏗️ Built with

- Next.js 14 (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- Framer Motion
- Google Gemini API
- Vercel
